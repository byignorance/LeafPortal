# SVG Export Pending

이 폴더는 실제 LEAF 아이콘 SVG 파일을 저장하는 위치입니다.

예정 파일명 예시:

- `leaf-logo.svg`
- `observation.svg`
- `path.svg`
- `alert.svg`
- `open-gate.svg`
- `footprints.svg`

현재 상태:

- `code.html` 기반 아이콘 세트는 일부 저장 완료
- Figma 커스텀 원본 세트는 아직 미생성

미생성 사유:

- Figma MCP 추가 호출 한도 초과
- MCP가 제공한 자산 URL은 외부 다운로드 시 `404`

다음 작업 시 이 폴더에 실제 SVG를 저장하면 됩니다.
